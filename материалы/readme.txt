1. Состав архива
--------------------------------------------------------------------------------------------

Электронный архив включает в себя следующие папки и файлы (вложенность папок и файлов
показана отступами):

  * lesson<номер урока>     - папка с результатами выполнения всех практических упражнений,
                              присутствующих в составе урока с соответствующим <номером>:

      - !sources            - папка с исходными материалами, необходимыми для выполнения
                              практических упражнений в текущем уроке;

      - data                - папка с дампами базы данных MongoDB todos.todos.json и
                              users.todos.json;

      - console             - папка с проектом простейшего консольного приложения конвертора
                              величин, написанного на уроке 1;

      - simple-web          - папка с проектом простейшего веб-приложения, написанного на
                              уроке 2;

      - express-web         - папка с проектом традиционного веб-приложения списка
                              запланированных дел, разрабатываемого на протяжении
                              уроков 3-16;

      - express-web-backend - папка с проектом бэкенда двухзвенного веб-приложения списка
                              запланированных дел, написанного на уроке 17;

      - react-web-frontend  - папка с проектом фронтенда двухзвенного веб-приложения списка
                              запланированных дел, разрабатываемого на протяжении
                              уроков 18-19;

 * readme.txt - этот текстовый файл.



--------------------------------------------------------------------------------------------
2. Развертывание веб-приложений
--------------------------------------------------------------------------------------------

Предполагается, что исполняющая среда Node.js, СУБД MongoDB и программа MongoDB Compass уже
установлены.

  1. Извлечь из архива нужную папку со всем ее содержимым.

  2. При необходимости - создать базу данных todos с коллекциями todos и users и
     импортировать в них дампы todos.todos.json и users.todos.json соответственно.
     Оба дампа хранятся в папке data.

  3. Запустить консоль.

  4. В консоли выполнить переход в извлеченную из архива папку.

  5. Выполнить установку зависимостей, отдав в консоли команду:

          npm install

     Через некоторое время все необходимые библиотеки, включая Express и React,
     будут установлены.

  6. Запустить приложение, отдав команду - в зависимости от разновидности приложения:

      - консольное приложение:

            npm run convert <преобразуемая величина>

      - традиционное веб-приложение или бэкенд:

            npm run serve

      - фронтенд:

            npm run start

  7. Запустить веб-обозреватель и открыть веб-приложение, набрав интернет-адрес -
     в зависимости от разновидности приложения:

      - традиционное веб-приложение:

            http://localhost:8000/

      - фронтенд:

            http://localhost:3000/
